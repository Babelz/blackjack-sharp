blackjack-sharp
======

CLI blackjack implemented in C#

## Dependencies:
* .NET core 3.1 
* https://dotnet.microsoft.com/download/dotnet-core/3.1

## To build:
* cd src
* dotnet build

## To run tests:
* cd src
* dotnet build
* dotnet test

## To run:
* cd src
* dotnet run --project Blackjack-Sharp/Blackjack-Sharp.csproj
